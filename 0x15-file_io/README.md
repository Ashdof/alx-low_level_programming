# C - File I/O

In this lesson, we explore file descriptors in C

## Meaning of File Descriptor
A file descriptor is a process-unique identifier or handle for a file or other input/output resource, such as a pipe or network socket [Wikipedia](https://en.wikipedia.org/wiki/File_descriptor). The file descriptor is an integer returned by an opened file. Any opened file has its own unique number. This file descriptor can be used to perform several operations on the file, including reading and writing to the file. [hacksland](https://hacksland.net/c-programming-file-descriptors).
