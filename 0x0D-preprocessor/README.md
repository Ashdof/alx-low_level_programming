0x0D. C - Preprocessor
This is the 14th lesson of the SE Training programme. We progress with
Preprocessor in C. The preprocessor is a macro processor that is used 
automatically by the C compiler to transform a program before actual
compilation. It is called a macro processor because it allows for the
definition of macros which are abbreviations for longer constructs.

The preprocessor provides the ability for the inclusion of header files,
macro expansions, conditional compilation, and line control.

* The files are compiled using gcc
* C files are written according to the C90 standard
* Tested on Ubuntu 20.04
